## TypeScript migration

This project has been migrated to TypeScript.

To build the app:

1. Install dependencies:

```bash
npm install
```

2. Compile TypeScript:

```bash
npm run build
```

3. Open `index.html` in your browser.

If you want automatic rebuilds during development, run:

```bash
npm run watch
```
